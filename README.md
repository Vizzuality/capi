# API Documentation

[Documentation can be found here.](http://docs.careusa.apiary.io)


## Dependencies

* Ruby 2.2.4
* Rails 5.0.0.beta3
* PostgreSQL

## Setting up                     

1- Clone the Application

`git clone git@github.com:Vizzuality/capi.git`

2- Run bundle
 
`bundle install`

3- Create your database (default username & password are both **postgres**)
 
`rails db:create`

4- Run migrations

`rails db:migrate`

5- Start your server

`rails server`
